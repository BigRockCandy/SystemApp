## 使用方法
配置easycom规则后，自动按需引入，无需`import`组件，直接引用即可。
```html
<template>
	<u-button text="按钮"></u-button>
</template>
```
请通过[快速上手](https://v2.uviewui.com/components/quickstart.html)了解更详细的内容
